Guidelines

Date Time Format

Dress code date and time format is currently based on British locale (en-gb). If in your application you use localization, we suggest to adapt the formats accordingly.

Date

Mon, 05 Sep 2016

05 Sep 2016

05 Sep

Sep 2016

Time

06:39

Date & Time

22 Dec 2016 20:04

22 Dec 20:04

Number Format

Dress code number format is currently based on British locale (en-gb). If in your application you use localization, we suggest to adapt the format accordingly. Depending on the number accuracy your application needs, feel free to use 0..N decimals.

Number

3,610,156.00

Percentage

36.40%

Currency

€1,439.34
